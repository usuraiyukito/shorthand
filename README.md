# 速記の簡単な勉強アプリ
画像または非表示ボタンを押すとそれぞれの表示非表示を切り替えが行われます

ペンマークまたはTTマークを押すと次の問題の時に速記を表示するかテキストを表示するか決めることができます。

NEXTのボタンを押すと非表示の物を表示し、もう一度押すと次の問題に行きます
